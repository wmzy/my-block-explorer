## 🔧 技术实现

### 组件架构
```
TopNavigation (主导航组件)
├── Logo (品牌标识)
├── SearchContainer (搜索区域)
│   ├── SearchInput (搜索输入框)
│   └── SearchButton (搜索按钮)
├── Controls (控制区域)
│   ├── RpcConfigButton (RPC配置按钮)
│   └── ChainSelector (链选择器)
│       ├── ChainButton (当前链显示)
│       └── ChainDropdown (链选择下拉)
│           ├── SearchInput (链搜索框)
│           └── ChainList (链列表)
└── RpcConfigModal (RPC配置模态框)
```

### 样式系统
- 使用 Linaria CSS-in-JS
- 响应式设计断点：768px
- 统一的设计令牌和颜色系统
- 无障碍设计支持

### 状态管理
- React hooks (useState, useEffect, useCallback)
- 本地状态管理，无需全局状态库
- 智能缓存和防抖优化
